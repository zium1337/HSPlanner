## New
- You can now add stars to your gear. Each star increased all stats (except x+ all skills affix) by **8%**.
- Added Satanic **Crystal/Gypsy's Prophecy** forging — apply one of 74 modifiers to any item with satanic+ rarity. Highlighted in red/pink.
- Added support for **"+x% Increased All Attributes"** affix.
- Support for **starring** gear.
- Allocate attributes faster — **Shift+Click** distributes 5 points, **Ctrl+Shift+Click** distributes all remaining. Same modifiers work on the − button to remove points.
- **Two-handed weapons** now lock the offhand slot. Equipping a 2H weapon clears any item in the offhand. The offhand slot stays disabled with a notice until the weapon is removed.

## Improved
- The **"+ Add affix"** system is now available only for common-rarity items (where it's used for runeword crafting). For satanic-tier items use Satanic Crystal, for heroic+ use Gypsy's Prophecy.
- Tooltip now support item images.
- Some gear affix changes to match with the game.

## Fixed
- Fixed **Manainfused Hour Glass** note.
- Weapon attack speed now correctly replaces the character's base 1.5 instead of stacking with it. A weapon with 1.0 attack speed gives 1.0 effective APS, not 2.25.
- **"+X% Increased All Attributes"** nodes on the passive tree were being applied as flat values (a "+5%" node gave +5 flat to every attribute). They now correctly multiply your attribute totals.
